IaaS - Infrastructure as a Service

  • Azure provides:
    • Servers and storage
    • Networking firewalls/security
    • Data center physical plant/building
  • You purchase/install/configure/manage your own software (OS, middleware, apps)
  • Example: Virtual Machine

PaaS - Platform as a Service

  • Azure provides:
    • Servers and storage
    • Networking firewalls/security
    • Data center physical plant/building
    • Development tools, database management, business analytics
    • Operating systems
  • You provide your applications
  • Examples: App Services, Azure SQL(?)

SaaS - Software as a Service

  • Azure provides:
    • Servers and storage
    • Networking firewalls/security
    • Data center physical plant/building
    • Development tools, database management, business analytics
    • Operating systems
    • Hosted applications/apps
  • You use the offering
  • Examples: SharePoint, SalesForce, YouTube

Quiz

For which offering am I not responsible for licenses?

SaaS - You pay for the SaaS and they handle licensing

Quiz

For which offering am I not responsible for security?

None - You have security concerns in all offerings

Quiz

What offering gives the most control?

IaaS - You have full control (and responsibility) of software and applications
